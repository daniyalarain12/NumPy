## 🧠 About This Repository

This repository contains **11 well-structured Jupyter Notebooks** that cover NumPy from the ground up. Every notebook is focused on a single topic with clean code, markdown explanations, and practical examples.

Whether you are a **complete beginner** or someone looking to **solidify your NumPy skills**, this repo is for you.

## 📚 Notebooks Overview

| # | Notebook | Topics Covered |
|---|----------|----------------|
| 01 | **Intro & Performance** | What is NumPy, Python Lists vs NumPy Arrays, execution time & memory comparison |
| 02 | **Array Attributes & Data Types** | `ndim`, `shape`, `dtype`, `itemsize`, `nbytes`, `astype()` |
| 03 | **Indexing, Slicing & Filtering** | 1D/2D/3D access, fancy indexing, boolean masking, loading from file, `any()` & `all()` |
| 04 | **Array Initialization** | `zeros()`, `ones()`, `identity()`, `full()`, `arange()`, `linspace()`, random arrays |
| 05 | **Broadcasting & Vectorization** | Scalar broadcasting, vector broadcasting, vectorized operations |
| 06 | **Array Manipulation** | `insert()`, `append()`, `concatenate()`, `delete()` on 1D and 2D arrays |
| 07 | **Repeating & Copying** | `repeat()` with axis control, `copy()` vs view |
| 08 | **Math, Linear Algebra & Statistics** | Arithmetic ops, trig functions, `matmul()`, `det()`, `inv()`, `trace()`, `min/max/mean/std/var` |
| 09 | **Reshaping & Flattening** | `reshape()`, `ravel()`, `flatten()`, view vs copy |
| 10 | **Stacking & Splitting** | `vstack()`, `hstack()`, `split()`, `hsplit()`, `vsplit()` |
| 11 | **Missing & Infinite Values** | `isnan()`, `isinf()`, `nan_to_num()`, handling NaN and Inf for data cleaning |

## 🎯 What You Will Learn

By going through all 11 notebooks in order, you will be able to:

- ✅ Understand why NumPy is faster and more memory-efficient than Python lists
- ✅ Create and inspect arrays of any shape and data type
- ✅ Index, slice, and filter arrays like a pro
- ✅ Initialize arrays using built-in NumPy functions
- ✅ Leverage broadcasting and vectorization for clean, loop-free code
- ✅ Manipulate arrays by inserting, deleting, and concatenating
- ✅ Perform linear algebra and statistical operations
- ✅ Reshape and flatten arrays for machine learning pipelines
- ✅ Stack and split arrays efficiently
- ✅ Handle missing and infinite values for real-world data cleaning
